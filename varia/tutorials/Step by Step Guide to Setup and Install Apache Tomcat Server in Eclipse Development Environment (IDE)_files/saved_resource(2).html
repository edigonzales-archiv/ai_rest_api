                  Борис Хрипко Jr Amarga • 4 months ago

                  I had the same problem.

                  1) Right click on the Tomcat on the tab "Servers" -> Properties -> Switch Location -> OK.

                  2) Double click on the Tomcat on the tab "Servers" -> On the tab "Server Locations" you must choose "use Tomcat installation" ->Save.

                  3) Deploy Path: webapps.

                  Ready.
